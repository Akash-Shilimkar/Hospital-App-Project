Hospital-App-Project

Built a hospital management API integration where staff can manage patient 
records, medicine inventory, and hospital data. Role-based login enables 
different types of access such as hospital staff and administrators.  

Prerequisites
eclipse IDE - Java EE
Java 17
Apache tomcat server
Postman to check API.

Download this project import in Eclipse IDE.
Then run the project as Java Application.
Hit the url on postman.
